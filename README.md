Программа содержит класс Employee, классы Manager, Worker и Brigadier которые его наследуют и класс Company. 
Каждый класс  который наследуется от класса Employee переопределяет его метод Work. Классы  Brigadier и Manager включают в себя дополнительные методы WorkersCheck и ToGiveATask соответственно.


Класс Company обобщён и при этом имеет ограничение для обобщения, для того чтобы открыть доступ к полям типов Employee внутри методов класса. Реализован паттерном синглтон и включает в себя весь требуемый функционал. Доступ к функционалу класса должен быть реализован через поле instance, доступ к которому в свою очередь можно получить через метод getInstance(). Класс включает в себя перегруженные операторы + и - с помощью которых происходит добавление и удаление сотрудников из общего хранилища EmployeeBase. С помощью метода расширения isExist можно проверить наличие экземпляра сотрудника в списке сотрудников фирмы. Метод amountEmployeesOfType возвращает количество сотрудников определённого типа, для определения нужного типа в метод передаётся экземпляр сотрудника. Методы getWorkers, getBrigadiers, getManagers возвращают последовательности сотрудников соответстующего типа. Метод расширения outputAll выводит в консоль имя каждого сотрудника фирмы.

В методе Main класса Program приведены примеры использования вышеупомянутых классов и их функционала. Присутствуют дополнительные конструкции сопутствующие проверке выполнимости функционала методов
